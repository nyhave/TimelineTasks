# TimelineTasks

TimelineTasks is a static product prototype for planning life events with template-based tasks and timeline deadlines.

## Current status

- ✅ Interactive front-end prototype in `index.html`
- ✅ Project templates, task lists, progress tracking, onboarding screens
- ❌ Production backend and deployment stack are not configured in this repository

## Firebase support

**Not yet.** There is currently no Firebase integration (no Auth, Firestore, or Hosting config) in this repo.
